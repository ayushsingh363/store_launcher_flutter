Launches the Google Play Store to the app's details page.

If packageName is provided, the method will use that to open the app page in the Play Store. If not, the method will fetch the current app's package name using PackageInfo.

Returns true if the Play Store was successfully launched, and false otherwise.

This method is only applicable for Android platforms. If the method is called on any non-Android platform, it will return false.

Example usage:

bool success = await StoreLauncher.launchPlayStore();

packageName - The optional package name of the app to be launched. If not provided, it will default to the current app's package name.

Implementation

static Future<bool> launchPlayStore({String? packageName}) async {
  try {
    if (!Platform.isAndroid) {
      return false;
    } else {
      PackageInfo packageInfo = await PackageInfo.fromPlatform();
      String package = packageName ?? packageInfo.packageName;
      final url = Uri.parse("market://details?id=$package");
      return await launchUrl(
        url,
        mode: LaunchMode.externalApplication,
      );
    }
  } catch (e) {
    print(e);
    return false;
  }
}
